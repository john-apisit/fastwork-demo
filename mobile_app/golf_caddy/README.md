# CaddyLink — Golf Caddy Marketplace Demo

แอปพลิเคชัน demo สำหรับเชื่อมต่อนักกอล์ฟกับแคดดี้ในประเทศไทย (Mobile-size web app)

## เทคโนโลยี

- Vue 3 + TypeScript
- Tailwind CSS
- Pinia + Vue Router
- Google Fonts (Cormorant Garamond + DM Sans)

## เริ่มต้นใช้งาน

```bash
yarn install
yarn dev
```

เปิดเบราว์เซอร์ที่ `http://localhost:5173`

## ฟีเจอร์ Demo

- สมัครสมาชิก (นักกอล์ฟ / แคดดี้)
- Top 10 แคดดี้ยอดนิยม
- เลือกวันที่ / สนาม / กรองแคดดี้
- ปลดล็อกโปรไฟล์และจองด้วยระบบเครดิต
- รับ/ปฏิเสธคำขอจอง (โหมดแคดดี้)
- แชทหลังยืนยันการจอง
- รีวิวหลังจบรอบ
- เติมเครดิต (QR / บัตร / E-Wallet)
- Knowledge Center (วิดีโอ YouTube)
- ติดต่อแอดมิน
