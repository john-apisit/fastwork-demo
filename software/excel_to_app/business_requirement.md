ตัวอย่างผลงาน web application 

ระบบงาน excel จากของเดิม ระบบลงงานหลังบ้าน

บรีฟงาน
 ต้องการให้คนมาช่วยจัดการข้อมูลใน Excel จากงานเดิมที่เรามีให้เป็นระบบและมีการลิงก์จากหน้างานหนึ่งไปอีกหน้างานหนึ่งโดยขั้นตอนในระบบของเอกซ์เซลเดิมและเป็นไปได้อาจจะพัฒนาในการใช้แอปพลิเคชั่นต่อไป และอยากให้เป็นคนที่สามารถมาดูงานที่หน้างานได้เพื่อจะได้อธิบายอย่างชัดเจนและเห็นภาพไปพร้อมกันค่ะ ซึ่งจะคล้ายๆกับการทำงานของการขนส่งสินค้าคล้ายๆแฟลชคล้ายๆไปรษณีย์ไทยแต่ของเราเป็นการขนส่งรถยนต์ไปตามดีลเลอร์ต่างๆทั่วประเทศไทย


เทคโนโลยี
- Vue
- TypeScript
- Tailwind CSS
